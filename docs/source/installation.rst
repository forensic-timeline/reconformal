Installation
============

You can easily install `reconsadfc` via PyPI or directly from the source code for development.

From PyPI (Recommended)
-----------------------

Once published, simply run:

.. code-block:: bash

   pip install reconsadfc

From Source (For Development)
-----------------------------

Clone the repository and install it in editable mode:

.. code-block:: bash

   git clone https://github.com/forensic-timeline/reconsadfc.git
   cd temp-reconformal
   pip install -e .[test]

System Requirements
-------------------
- Python 3.9+
- pandas >= 1.5
- matplotlib >= 3.6
